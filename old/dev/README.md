# Инструкция по разработке с докером
## Сборка образа
Для запуска процесса сборки докер образа выполнить:
```bash
./dev_build.sh
```
## Запуск докера
После того как был собран докер образ можно запустить докер:
```bash
./dev_run.sh
```

## Работа в докере
При первом запуске докер образа проект не будет собран. Для сборки всего проекта можно выполнить следующую команду:
```bash
ws_build.sh # или ws_build.sh --all
```

Также можно перейти собрать конкретный модуль командой:
```bash
ws_build.sh --this
```

Для сборки модулей в папке использовать следующую команду:
```bash
ws_build.sh --folder
```

После сборки всего проекта можно запускать модули. На данный момент можно запустить симулятор и стек навигации.
Запуск симулятора:
```bash
ws_run.sh --sim
```
Для запуска стека навигации:
```bash
ws_run.sh --planning
```
Так же можно сделать соурс текущего энвайромента:
```bash
. ws_source.sh
```
